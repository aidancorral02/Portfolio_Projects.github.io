NASA Project

My final capstone project was intended for my team and I to develop an algorithm that can be incorporated in drone technology, which can be used to identify obstacles in areas that have been surveyed by aerial LIDAR. We used USGS Aerial LIDAR and DALES Semanatic Segmentation data for our calculations. The methods used for the development are a filtering algorithm called "The Cloth Simulation" which was done in R Studio, Point Segmentation/K Means Clustering done in Python using libraries such as Scikit-learn, numpy, pandas, laspy, and others as well for the clustering algorithm, and Excel to export the clusters points of radius, max height, longitude, latitude, and height.
